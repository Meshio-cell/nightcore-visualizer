How to Use

Nightcore Visualizer Guide / ナイトコアビジュアライザー使い方ガイド

What is this? / これは何？

Nightcore Visualizer is a free browser-based anime-style audio visualizer.

Nightcore・AMV・VTuber・LoFi・TikTok・YouTube Shorts向けの、 ブラウザだけで動作する無料オーディオビジュアライザーです。

Quick Start / クイックスタート

1. Upload Music

Upload your audio file using the file picker.
音楽ファイルを読み込みます。

2. Customize Visualizer

Change waveform, colors, glow, background, and sensitivity.
波形・カラー・Glow・背景・感度などを調整できます。

3. Export Video

Press Auto Export or Auto Transparent.

Auto Export： 通常背景付き動画を書き出します。
Auto Transparent： 背景透過WebMを書き出します。

Transparent Export / 透過書き出し

Transparent Export allows you to overlay waveforms on videos in editing software such as:

  • CapCut
  • Adobe Premiere Pro
  • OBS Studio
  • After Effects

波形だけを動画上に重ねることができます。

Recommended Browser / 推奨ブラウザ

  • Google Chrome（Recommended）
  • Safari（Some limitations）
  • Firefox（Limited support）

Export Quality / 書き出し品質

  • High：60fps / 16Mbps
  • Medium：60fps / 8Mbps
  • Low：30fps / 4Mbps

Tips / おすすめ設定

  • Bars mode is the smoothest.
  • Glow 8〜12 is recommended.
  • 56 bars gives good performance.
  • Use Chrome for best export stability.

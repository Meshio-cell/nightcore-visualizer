# Nightcore Visualizer

Free anime-style audio visualizer for Nightcore, AMV, VTuber, LoFi, and music content creators.  
ナイトコア・AMV・VTuber・LoFi・音楽系動画向けの、無料ブラウザ型オーディオビジュアライザーです。

---

## Features / 主な機能

- Multiple visualizer modes / 複数のビジュアライズモード
  - Linear Bars
  - Smooth Wave
  - Mirror Wave
  - Glowing Dots
  - Circle Ring
- Real-time audio reaction / 音楽に合わせたリアルタイム反応
- Transparent WebM export / 透過WebM書き出し対応
- Auto Export mode / Auto Export機能
- Warmup rendering system / ウォームアップ描画システム
- High / Medium / Low quality export presets / 高画質・中画質・低画質 書き出し
- Glow and bass boost controls / Glow・低音ブースト調整
- Custom colors and themes / カラー・テーマ変更
- Fullscreen mode / フルスクリーン対応
- Browser-based, no install required / ブラウザのみで動作、インストール不要

---

## Export Quality / 書き出し品質

|---|---:|---:|

## How to Use / 使い方

1. Upload an audio file / 音楽ファイルを読み込む
2. Customize the visualizer / ビジュアライザーを調整
3. Press Auto Export or Auto Transparent / Auto Export または Auto Transparent を押す
4. Download the generated WebM video / WebM動画をダウンロード
